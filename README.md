# **Online Chat**

---
### Description : 
 an `API` build with `django rest framework ` 

--- 
### Features  :

+ sending sms to register 
+ using `Rabbitmq` as a message broker 
+ using `redis` as a result backend 
+ using `celery`

--- 
### How To RUN :
+ run `docker compose up --build`
+ open your brower at `http://127.0.0.1:8000`
